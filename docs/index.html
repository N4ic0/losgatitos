Motel Los Gatitos

Sistema de gestión hotelera — Reservas, habitaciones, tarifas y más

Laravel 13 PHP 8.4 MySQL 8.0 Docker Tailwind CSS Alpine.js

Stack Tecnológico

🐘 Laravel 13.x ⚡ PHP 8.4 🐬 MySQL 8.0 🐳 Docker + Nginx 🎨 Tailwind CSS 3 ⚙️ Alpine.js 📦 Vite 🔐 Laravel Breeze 🇨🇱 RUT chileno

Módulos del Sistema

🌐 Landing Page

Sitio web público con información del motel

  • Hero con video de fondo
  • Galería de habitaciones
  • Promociones activas
  • Formulario de reserva con cálculo de precio en vivo
  • Sección de contacto con Google Maps

🔐 Autenticación

Login y registro con Laravel Breeze

  • Login con email y contraseña
  • Roles: administrador / recepcionista
  • Recuperación de contraseña
  • Verificación de email
  • Perfil de usuario

🛏️ Admin Panel

Panel de administración completo

  • Dashboard con estado de habitaciones en tiempo real
  • CRUD de habitaciones, reservas, tarifas
  • Gestión de promociones y feriados
  • Asignación y liberación de habitaciones
  • Cobro de horas adicionales

💰 Tarifas Dinámicas

Precios según día y categoría

  • Tarifas diferenciadas: DJ / viernes / sábado / víspera
  • Recargo por tercera persona
  • Horas adicionales
  • Promociones por fecha
  • Detección automática de feriados
